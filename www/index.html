hafroAssmt

The hafroAssmt package has migrated from R-Forge to a local server at Hafro:

  • Source repository: svn://svn.hafro.is/r
  • Package repository: http://r.hafro.is

The purpose of the migration was to test a new R package development and distribution system at Hafro. It turned out quite successful and is now used for several Hafro-specific R packages.

The last R-Forge release of hafroAssmt (version 1.7-1 from 2012-05-15) is available from the tags directory.

The current version is available within Hafro by running: 

install.packages("hafroAssmt", repos="http://r.hafro.is")
